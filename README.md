# Generador de Facturas - ACUÑA LOLY DIKSON YAMPIER

Aplicación web para generar facturas en formato PDF.

## Características

- Generación de facturas electrónicas
- Cálculo automático de IGV (18%)
- Exportación a PDF
- Interfaz moderna y fácil de usar

## Uso

Simplemente abre el archivo `code.html` en tu navegador o visita la página en GitHub Pages.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- jsPDF
- html2canvas
